# iHR 项目

## 0）一句话产品（低频更新）

iHR：B端发起批量初筛 → C端候考室设备检测 → 语音面试（可视化声波）→ AI 输出结构化证据报告 → B端全屏阅卷决策。

---

## 1）产品骨架（低频更新）

### B端（侧边栏 + 全屏阅卷）

* 入口：浏览器侧边栏（Side Panel）
* 发起：拖拽上传PDF → 意图卡片 → 标签云微调（替代表单）
* 进度：卡片流 + 状态流转 + 全链路日志时间轴
* 阅卷：新标签页全屏（左PDF / 右报告；证据点击联动PDF定位）
* 决策：Proceed / Follow-up / Hold（三态路由）

### C端（候考室 Gate + 通话页）

* 信任落地页：Logo + 15分钟预期
* 候考室 Gate：麦克风无输入 → 按钮置灰 → 不可进入
* 通话页：Visual Audio（声波/波纹）+ 辅助信息卡（如代码截图）
