## 2. 产品定位

### 一句话
> **你的 AI 招聘助理**

### 一段话（适用于产品介绍、Chrome 商店详情）
> Ailin 以 Chrome 扩展形式在你熟悉的招聘网站侧边栏中工作。不离开 Boss 直聘、智联、猎聘等页面，即可完成简历深度解析、AI 自动初面与结构化评估报告——显著缩短从「看到简历」到「做出决策」的时间。

### Chrome 插件 description（132 字符内）
> 在招聘网站侧边栏实时解析简历亮点，一键发起 15 分钟 AI 初面，自动生成结构化评估报告

## 3. 三大核心功能

每个功能有三个长度版本，各触点按需取用：

### 功能 1：侧栏伴随

| 版本 | 文案 |
|------|------|
| **标题**（4 字） | 侧栏伴随，无需切换 |
| **一句话** | 浏览简历时，侧边栏实时提取候选人核心亮点与经历风险点 |
| **详细版** | 在 Boss 直聘、智联、猎聘等招聘网站浏览简历时，侧边栏实时提取候选人的核心亮点与经历风险点。无需来回切换系统，信息就在手边。 |

> **注**：此功能在 Chrome 侧边栏欢迎消息中**不展示**（功能价值感弱，侧边栏空间有限）。仅出现在 WelcomeView、productIntro 等完整产品介绍场景。

### 功能 2：AI 初面

| 版本 | 文案 |
|------|------|
| **标题**（6 字） | 15 分钟全自动初面 |
| **一句话** | 一键发送面试邀约，AI 代你完成多轮语音面谈 |
| **详细版** | 一键发送面试邀约，AI 代你完成多轮语音面谈。还原真实沟通语境，深挖简历背后的专业能力与求职动机。 |
| **extensionMode 极简版**（侧边栏窄屏） | AI 全程主持，多轮语音面谈 |

### 功能 3：结构化报告

| 版本 | 文案 |

| **标题**（8 字） | 结构化报告，拒绝模棱两可 |

## 7. 术语规范：初面 vs 初筛

**核心规则：产品动作描述用"初面"，HR 工作流程阶段描述可用"初筛"。**

### 叙事框架

```
痛点场景（"初筛"在此有效）：初筛环节耗时费力
产品动作（统一用"初面"）：AI 替你做初面（15 分钟语音约谈）
产出结果：结构化评估报告
```

### 允许保留"初筛"的场景（状态标签 / 痛点描述）

| 场景 | 示例 | 说明 |
|------|------|------|
| 候选人进度标签 | `初筛通过` | 描述候选人所处的招聘阶段 |
| 报告卡标题 | `初筛报告` | 报告是整个初筛阶段的产出 |
| 方案卡标签 | `初筛方案` | 覆盖整个初筛流程的方案 |
| 效率数据指标 | `初筛时间节省` | 描述阶段效率提升 |
| 痛点描述 | `初筛环节耗时费力` | 描述 HR 遇到的问题 |

### 禁止用"初筛"的场景

- 描述 Ailin 产品能力时：❌ "让 AI 替你做初筛" → ✓ "让 AI 替你做初面"
- CTA 按钮：❌ "开始初筛" → ✓ "开始初面"
- 欢迎语中描述 Ailin 做的事：❌ "初筛全流程" → ✓ "初面全流程"

## 8. Extension SidePanel 未登录态文案规范

Chrome 侧边栏在两种情境下触发，文案差异化处理。Chrome 原生头部已显示"Ailin AI 招聘助理"，无需重复品牌头。

### 情境一：用户正在浏览候选人简历（`browserContext === 'resume'`）

```
[情境感知 Banner，淡蓝背景 + 脉冲绿点]
  我注意到您正在查看一份候选人简历
  登录后，我立即为您开始分析

[价值点]
  ① 15 分钟全自动初面  /  AI 全程主持，多轮语音面谈
  ② 结构化报告，拒绝模棱两可  /  量化评分报告，带红笔批注

[主 CTA] 登录，立即初面这位候选人
```

### 情境二：其他页面（`browserContext === 'empty'`）

```
[主标语] 让 AI 替你做初面
[副标语] 15 分钟完成一次专业初面

[价值点]
  ① 15 分钟全自动初面  /  AI 全程主持，多轮语音面谈
  ② 结构化报告，拒绝模棱两可  /  量化评分报告，带红笔批注

[主 CTA] 登录，立即开始
```

## 9. 禁止清单

- "IHR-NEXUS"（内部代号）
- "Chrome Extension"（英文技术术语）
- "授权连接"（改为"一键激活"）
- "检测到登录态"（技术术语，改为"已识别你的身份"）
- "token"、"cookie"、"JWT"（任何技术实现细节）
- 单独使用"艾琳"（必须是 Ailin 或 Ailin（艾琳））
- 在产品动作描述中用"初筛"（统一用"初面"，详见第 7 节）
- manifest description 中写平台名"智联招聘"（用通用词"招聘网站"）
- "随时为您服务"（不是定位语，header 副标题统一用"你的 AI 招聘助理"）
